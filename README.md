\# Django Blog \& Meal Tracking App



A Django web application that includes a basic blog system and a meal tracking system.



\## Features



\- Blog post CRUD operations

\- Meal create, update, delete and detail pages

\- Search meals by name

\- Filter meals by meal type

\- Daily calorie and macro calculation

\- Basic template structure and custom CSS



\## Technologies Used



\- Python

\- Django

\- HTML

\- CSS

\- SQLite



\## How to Run



```bash

git clone https://github.com/cmdnzgk/Django-blog-meal-app.git

cd Django-blog-meal-app

python -m venv venv

venv\\Scripts\\activate

pip install django

python manage.py migrate

python manage.py runserver

Then open: http://127.0.0.1:8000/

Project Purpose

This project was developed as part of my learning process while studying Python and Django.
It demonstrates basic Django project structure, CRUD operations, forms, templates, search, filtering and simple calorie/macro calculations.
